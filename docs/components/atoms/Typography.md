# Typography

Source: `src/components/atoms/Typography/Typography.tsx`

Exports: `Typography`

Opis:
- Komponent typograficzny przyjmujący wariant (`h1`, `h2`, `h3`, `body`, `small`) i renderujący odpowiedni element.

Użycie:
```tsx
import { Typography } from 'src/components/atoms/Typography/Typography';
<Typography variant="h1">Tytuł</Typography>
```
