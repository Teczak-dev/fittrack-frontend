# Fittrack - Frontend solution

Ta aplikacja umożliwia śledzenie treningów, dodawaj, edytuj i analizuj swoje dotychczasowe postępy.

## Główne funkcje:
    - Tworzenie i logowanie konta,
    - dodawanie, przeglądanie i usuwanie swoich treningów,
    - postępy ćwiczeń z dnia, tygodnia, miesiąca,
    - analizowanie postępów( czas, kalorie, typ ćwiczeń ),
    - utrzymuj streak ćwiczeń,
    - znajomi, dodawanie ich oraz śledzenie ich poczynań ( steak ),
    - dodanie motta,
    - wyszukiwanie i filtrowanie treningów,
    - jasny i ciemny motyw,
    - pełna responsywność na urządzenia przenośne,
